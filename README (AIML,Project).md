# Student Performance Prediction using Machine Learning (Python)
This is my first aiml project it is a Simple project in which we predict **Students performance** using the number of hours studied by each student then thier performance is categorized into pass and fail 

## Project Objective:-
1)To understand the working of Linear Regression

2)To predict student performance based on study hours

3)To classify output into PASS / FAIL

## Machine Learning Concept Used
Linear Regression:-
Linear Regression is a supervised Machine Learning algorithm used to predict continuous values. It finds the relationship between:

Independent Variable (X) → Study Hours

Dependent Variable (Y) → Marks Obtained

The equation used is:

[
y = mx + c
]

Where:

y = predicted marks

x = study hours

m = slope

c = intercept

## How the Project works
1)The dataset is loaded from data.csv

2)The data is trained using Linear Regression

3)The user enters study hours

4)The model predicts expected marks

5)The result is classified:-

✅ PASS → If marks ≥ 40

❌ FAIL → If marks < 40
